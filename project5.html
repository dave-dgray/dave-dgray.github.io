Car Crash Heritage Repairs

A Grade 2, late medieval timber frame house. I previously assisted the present owner in attaining listed building consent approval and clearing planning conditions, as part of alteration works to return the main part of the property to a private residential dwelling. Part of the property was damaged by a car impact. I have been working with the client and loss adjuster in attaining the Listed Building Consent approval for the reinstatement works. Project reference code 126.
